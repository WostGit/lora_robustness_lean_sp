# LoRA Robustness Lean/SP Artifact

This repository contains a reviewer-oriented artifact for the LoRA robustness framework and privacy interactions.

Start with:

```bash
bash scripts/reviewer_quickstart.sh
```

The GitHub Actions workflow runs the reviewer quickstart and writes generated outputs under `logs/`.

This artifact is honest about scope: it contains Lean specifications, beginner documentation, and a deterministic toy reproducibility workflow. It does not claim to be a complete proof of all LoRA robustness results or a full BERT/GLUE reproduction.
